# F1 Cost Cap Financial Model — McLaren Racing (2025)

This project is a comprehensive financial and strategic model built to simulate how a Formula 1 team operates under the FIA cost cap.

It combines financial modeling, scenario analysis, and performance simulation to evaluate how budget allocation decisions impact on-track outcomes.

---

## Project Objective

To design a realistic model that:

- Allocates a fixed FIA cost cap ($135M) across key departments  
- Tracks season-long expenditure across 24 races  
- Evaluates trade-offs between performance, reliability, and risk  
- Links financial investment to racing performance metrics  

---

## Key Features

### 1. Cost Cap Allocation Model

- Budget allocation across:
  - Aerodynamic R&D  
  - Manufacturing & Production  
  - Race Operations & Logistics  
  - Trackside Engineering  
  - IT & Simulation  
  - Contingency Reserve  

- Scenario-based planning:
  - Base Case (Balanced)
  - Aggressive Development
  - Operations Focus  

---

### 2. Race-by-Race Burn Tracker

- 24-race season simulation  
- Differentiation between:
  - Standard vs Sprint weekends  
  - European vs Flyaway races  

- Tracks:
  - Cumulative spend  
  - Remaining cap  
  - Burn rate status  

---

### 3. Performance Modeling (PROCS Index)

A custom-built **Performance Return on Cap Spend (PROCS)** index that:

- Translates financial allocation into performance metrics:
  - Lap time improvement  
  - Reliability score  
  - Pit stop efficiency  
  - Simulation accuracy  

- Evaluates efficiency of spending decisions  

---

### 4. Sensitivity Analysis

- Focus on **Aerodynamic R&D investment**
- Measures:
  - Impact on lap time  
  - Trade-offs with contingency reserve  
  - Risk vs performance gain  

---

### 5. Competitor Benchmarking

Comparison with top Formula 1 teams:

- Red Bull  
- Ferrari  
- Mercedes  
- Aston Martin  

Metrics used:

- Points per $ spent  
- Wins per $10M  
- Reliability score  

---

### 6. Strategic Insights

Key findings from the model:

- McLaren is efficient but underinvesting in Aerodynamics  
- Aggressive development improves lap time but reduces contingency buffer  
- Operations-focused strategy improves reliability but limits peak performance
